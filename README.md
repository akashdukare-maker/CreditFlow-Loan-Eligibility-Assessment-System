## 📌 Overview

CreditFlow is a Machine Learning-based Loan Eligibility Assessment System that predicts whether a loan application should be **Approved** or **Rejected** based on an applicant's financial, demographic, and credit-related information.

The goal is to assist financial institutions in making **faster, more accurate, and unbiased lending decisions** while reducing manual effort and financial risk.

---

## 🚀 Problem Statement

Traditional loan approval processes often involve manual verification, which can be:

- ⏳ Time-consuming
- ⚠️ Inconsistent
- 🎯 Prone to human bias

CreditFlow leverages Machine Learning to automate loan eligibility assessment using historical applicant data.

---

## 📊 Dataset Features

| Feature | Description |
|----------|------------|
| Applicant_Income | Monthly income of applicant |
| Coapplicant_Income | Monthly income of co-applicant |
| Employment_Status | Salaried / Self-Employed / Business |
| Age | Applicant age |
| Marital_Status | Married / Single |
| Dependents | Number of dependents |
| Credit_Score | Credit bureau score |
| Existing_Loans | Existing active loans |
| DTI_Ratio | Debt-to-Income ratio |
| Savings | Savings balance |
| Collateral_Value | Value of collateral |
| Loan_Amount | Requested loan amount |
| Loan_Term | Loan duration |
| Loan_Purpose | Purpose of loan |
| Property_Area | Urban / Semi-Urban / Rural |
| Education_Level | Education qualification |
| Employer_Category | Government / Private / Self |
| Loan_Approved | Target Variable |
